Takeaway 1 — Evidence turns “mythic time” into history

  • This period shows archaeology confirms written claims.
  • This matters because oracle bones reveal state religion.
  • This matters because Anyang anchors Shang chronology.

Takeaway 2 — Kings ruled by ritual as much as force

  • This period shows kings mediated ancestors and Di.
  • This matters because divination organized state decisions.
  • This matters because bronze vessels signaled elite power.

Takeaway 3 — Zhou legitimation was conditional, not permanent

  • This period shows Heaven can withdraw legitimacy.
  • This matters because virtue explains dynastic change.
  • This matters because rebellion can be morally framed.

Takeaway 4 — Early China was plural, not monolithic

  • This period shows regional cultures shaped “China.”
  • This matters because Sanxingdui complicates the core.
  • This matters because Zhou labels built boundaries.

Part II — Common Mistakes to Avoid

Check each misconception you will actively avoid on the exam.
Treating the Shang as “myth” like Xia (Shang is verified).
Assuming the Mandate of Heaven = permanent divine right.
Thinking oracle bones are “just religion,” not governance.
Reducing bronze to weapons (ignore ritual function).
Forgetting only the king performed oracle divination.
Reading “barbarians” as objective ethnicity (it’s a Zhou category).
Assuming Sanxingdui was Shang-controlled or “primitive.”
Misplacing the 771 BCE crisis and the move to Luoyang.
