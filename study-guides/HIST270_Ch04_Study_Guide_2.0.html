Part I — The 4 Big Takeaways (Short Answers — No Paragraphs)

Keep these four in your head while you do Parts II–VI. If you can’t connect an answer to a takeaway, you’re probably memorizing trivia.

Takeaway 1 — Collapse → fragmentation → local power

  • This period shows rebellions become political accelerants.
  • This matters because local forces become autonomous.
  • This matters because “unity” is historically fragile.

Takeaway 2 — North/South division reshaped identity

  • This period shows “Chinese” was not a fixed label.
  • This matters because conquerors ruled multiethnic states.
  • This matters because southern refuge built new elites.

Takeaway 3 — Northern Wei “sinicization” was statecraft

  • This period shows policy can remake elite culture.
  • This matters because reforms produced frontier backlash.
  • This matters because land/tax systems required registration.

Takeaway 4 — Buddhism became a mass institution

  • This period shows monasteries offered meaning and services.
  • This matters because pilgrimage linked China to India.
  • This matters because art sites spread Buddhist devotion.
